# 📘 Multi-Agent CDSS

Система підтримки прийняття клінічних рішень (СППКР) на базі фреймворку LangGraph, яка симулює роботу медичного консиліуму за допомогою AI-агентів.

---

## 📌 Загальна інформація

- **Тип проєкту**: Веб-додаток
- **Мова програмування**: Python
- **Фреймворки / Бібліотеки**: LangGraph, LangChain, Streamlit

---

## 🧠 Опис функціоналу

- 📋 Введення та текстовий аналіз складних анамнезів пацієнтів
- 🔀 Перемикання в реальному часі між трьома топологіями графів (Independent, Collaborative, Orchestrator)
- 👥 Симуляція міжагентного консиліуму за участю трьох ролей: Діагноста, Лаборанта та Фармацевта

---

## 🧱 Опис основних файлів

| Файл             | Призначення                                  |
| ---------------- | -------------------------------------------- |
| `app.py`         | Точка входу додатка                          |
| `state.py`       | Визначення моделей додатку                   |
| `agents/`        | Реалізація та ініціалізація рольових агентів |
| `prompts/`       | Шаблони поведінки для кожного AI-агента      |
| `architectures/` | Реалізація топологій міжагентної взаємодії   |

## ▶️ Як запустити проєкт "з нуля"

### 1. Встановлення інструментів

- Python v3.10+

### 2. Клонування репозиторію

```bash
git clone https://github.com/frant1x/mas_cdss.git
cd mas_cdss
```

### 3. Створення віртуального середовища

```bash
python -m venv venv
venv/scripts/activate
```

### 4. Встановлення залежностей

```bash
pip install -r requirements.txt
```

### 5. Створення `.env` файлів

```
GOOGLE_API_KEY=your_api_key
```

### 6. Запуск

```bash
streamlit run app.py
```
